欢迎使用 marchen 前端模板

开发环境 node + express
预配置 gulp + normalize.css + lesshat.less + marchen.less and more

直接下载使用:

FE-Template-0.0.8.zip

使用方法:

1.安装node:

到官方网站下载安装nodejs环境: nodejs官网

2.安装相关依赖:

进入命令行输入 npm install ,自动安装相关依赖

3.开启nodejs服务:

由于 node ./bin/www 开启node服务后,改变文件需要手动重启服务.不适合开发环境,所以推荐:

用supervisor或者node-inspector这2款插件来开启node服务,当文件改变后,将会自动重启node服务

4.运行gulp

依赖安装成功后,运行 gulp,自动编译less,coffee,js文件

gulp 任务有4个:

    gulp 默认编译less,coffee,js文件.不会打开浏览器同步工具,只是起到监听编译的效果

    gulp proxy 编译less,coffee,js文件后打开浏览器同步工具,监听文件改变后,自动刷新浏览器,代理模式,默认代理localhost:3000,需要请到gulpfile.js文件修改

    gulp server 编译less,coffee,js文件后打开浏览器同步工具,监听文件改变后,自动刷新浏览器,服务器模式,打开当前目录下文件

    gulp server-no-compile 静态服务器模式,默认代理 './' ,监视,js,css,views目录下文件改变后,自动刷新浏览器,需要请到gulpfile.js文件修改

4.恭喜 你已经完成了开发环境的搭建

开启后,试试直接用你喜欢的编辑器改变下views下的文件.将会直接看见效果了.(如果是用的gulp localhost任务的话)
